Contribute

TestDataMT is an open-source project and all contributions are welcome to assist with its development and maintenance.

Issues (bug and feature tracker)

Please report any bugs found, feature requests or other issues on TestDataMt GitHub tracker.

When creating a new issue, try following necolas’s guidelines.

Fork, patch and contribute code

Feel free to fork TestDataMT Git repository at GitHub for your own use and updates.

Contribute your fixes and new features back to the main codebase using GitHub pull requests.
