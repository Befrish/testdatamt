Project Modules

This project has declared the following modules:

Name Description
TestDataMT - Collection Utilities Specific additional collections e.g. special implementation of set of objects with an unique key
TestDataMT - Utilities General, frequently used utility classes
TestDataMT - Domain Domain objects for reading, evaluating and processing test data
TestDataMT - API API to run TestDataMT
TestDataMT - DSL German Source TestDataMT source for reading test data expressed as text input in german DSL (e.g. files)
TestDataMT - Builder Source TestDataMT source for reading test data of an data object builder
TestDataMT - Evaluation TestDataMT processors for verifing test data semantically
TestDataMT - Database Sink TestDataMT sink to export test data to database
TestDataMT - Systemtest Systemtests to test full roundtrips of running TestDataMT
TestDataMT - Benchmarks Benchmark tests in order to prove performance improvments
